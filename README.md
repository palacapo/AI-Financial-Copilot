# AI-Financial-Copilot
An AI-powered financial copilot built with n8n, LLMs and market data APIs that provides daily market analysis, news summaries, risk assessments and portfolio recommendations.

# AI Financial Copilot

An AI-powered financial assistant that automatically monitors financial markets and provides intelligent daily reports.

## Features

- Bitcoin & Crypto Market Analysis
- Fear & Greed Index
- Financial News Monitoring
- AI-powered Market Insights
- Telegram Notifications
- Portfolio Suggestions
- Risk Assessment
- Live Bitcoin Price
- Fear & Greed Index
- Crypto News Aggregation
- AI Market Analysis (Gemini 2.5 Flash)
- Telegram Notifications

## Tech Stack

- n8n
- Google Gemini
- CoinGecko API
- Fear & Greed API
- Telegram Bot API
- GitHub
